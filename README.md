# Multi-hazard-Simulation-Tool-v1.0
The Multi-Hazard Simulation Tool v1.0 is an ArcGIS (Pro) Python Script Toolbox for automated runout simulation and hazard exposure assessment in high mountain regions, integrating Rock/Ice Avalanche (RIA), Glacial Lake Outburst Flood (GLOF), and Landslide (LS) models into a normalized composite Exposure Index.
